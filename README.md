# shuttle-time-finder
Given your current location, calculates when the next UTM-UTSG shuttle bus is

# Dependencies
## Python
- pip install bs4
- pip install requests

## node.js
- npm 6.10.3
- do `npm install`

## Express

### APIs
- https://developers.google.com/web/fundamentals/native-hardware/user-location
